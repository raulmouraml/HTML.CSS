Texto pequeno

Podemos inserir um texto com letras miudas com a Tag sall. Exemplo: Vários contratos tem letras miudas no final.

Podemos aplicar um texto excluído com a Tag del. Exemplo: Texto excluído.

Texto inserido

O texto inserido ou texto sublinhado é inserido atravéz da Tag ins.

Exemplo: Complete a frase. O Palmeiras não tem mundial.

Texto sobrescrito

Para inserirmos texto sobrescrito ultizamos a Tag sup.

Exemplo: X 2+4
